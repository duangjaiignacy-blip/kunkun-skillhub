# SkillHub 功能介绍

SkillHub 是一个面向 Claude Code 与 Codex 的本机技能一体化管理台，基于 Honeycomb 工程搭建，采用 macOS 单机桌面应用形态，聚焦技能盘点、调用统计、优先级管理与安全清理。

## 核心功能

1. 技能目录总览
   - 自动扫描 Claude Code、Codex、共享技能源与插件缓存中的技能。
   - 以卡片墙展示技能名称、来源、描述、健康状态与安装形态。
   - 支持按 Claude、Codex、高优先级、未使用、异常技能筛选。

2. 调用次数统计
   - Claude 侧读取显式 Skill 调用记录，标记为精确统计。
   - Codex 侧通过 function_call、读命令与 SKILL.md 路径组合识别真实读取记录，标记为推断统计。
   - 展示最近使用日期与调用热度，方便判断常用技能和长期未使用技能。

3. 优先级管理
   - 支持将技能标记为高优先级、普通或低优先级。
   - 用户标签独立保存在本地应用数据目录，重新扫描不会覆盖手工标记。

4. 安全删除
   - 删除前先进行预检，区分真实目录、共享源软链、外部软链与断链。
   - 软链只断开链接节点，不跟随删除共享源。
   - 删除操作进入系统废纸篓，可恢复，避免误删真实技能源。

5. 健康检查基础能力
   - 识别缺失 frontmatter、断链、不可读等异常技能。
   - 已为后续日志体检、自动更新、主动提醒 hook 预留数据结构。

## 技术实现

- 工程基础：Honeycomb 工程
- 桌面框架：Tauri 2
- 扫描内核：Rust scanner-core
- 前端形态：零外部依赖静态页面
- 数据策略：技能源只读扫描，用户元数据单独持久化
- 安全策略：前端无文件系统权限，所有磁盘访问收敛到后端 command

## 当前状态

- 已完成扫描内核 CLI。
- 已完成桌面 MVP：技能目录、调用次数、优先级、删除预检与安全删除。

- 自动更新、日志错误检查、Web 桥与主动提醒 hook 属于后续阶段。
